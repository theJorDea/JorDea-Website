# JorDea Website

Personal developer portfolio migrated from a static GitHub Pages site to a Vercel-ready Next.js app.

## Stack

- Next.js App Router
- React
- TypeScript
- CSS custom properties
- `next/image` for local assets

## Local commands

```bash
npm install
npm run dev
npm run build
```

## Vercel import settings

- Framework preset: `Next.js`
- Root directory: `./`
- Build command: leave default (`next build`)
- Output directory: leave default
- Install command: leave default
- Environment variables: none required

Vercel will auto-detect the project from `package.json` and the `src/app` directory.
