# JorDea Website

Minimal personal portfolio for JorDea, focused on Junior ML/NLP engineering, PyTorch, RAG systems, Deep Learning and Audio ML.

## Stack

- Next.js App Router
- React
- TypeScript
- CSS custom properties
- Motion for measured UI animation
- Phosphor Icons for a single icon family
- `next/image` for local assets

## Local commands

```bash
npm install
npm run dev
npm run build
```

## Vercel import settings

- Framework preset: `Next.js`
- Root directory: `./`
- Build command: leave default (`next build`)
- Output directory: leave default
- Install command: leave default
- Environment variables: none required

Vercel will auto-detect the project from `package.json` and the `src/app` directory.
